# Sistema de Análisis Topohidrológico Automatizado con LiDAR

Scripts Python desarrollados como parte del Trabajo Final del Máster GIS de Esri España, edición 2025-2026.

## Descripción del proyecto

Sistema integral para automatizar el análisis topográfico e hidrológico de predios destinados al desarrollo inmobiliario en México. Procesa datos LiDAR crudos y genera los insumos cartográficos requeridos por las autoridades municipales, estatales y federales.

## Scripts incluidos

### GenerarPourPoints.py
Identificación automática de pour points (puntos de salida del agua) en un predio mediante análisis topológico de la red de drenaje. Resuelve el problema clásico de identificación manual subjetiva, distinguiendo entradas de salidas y agrupando puntos por arroyo.
